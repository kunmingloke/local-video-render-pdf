# Notice

This repository contains `local-video-render-pdf`, a personal Codex skill modified for local video workflows.

The workflow was inspired by and adapted from the video-to-PDF note-generation skills in:

https://github.com/wdkns/wdkns-skills

Credit goes to the original author(s) of `wdkns/wdkns-skills` for the original YouTube/Bilibili video rendering and Chinese lecture-note generation approach.

This version was modified for personal use by `kunmingloke` to support:

- local video files
- local and embedded subtitles
- Whisper fallback transcription
- locally extracted cover frames
- Windows/MiKTeX-oriented compilation checks
- fallback PDF rendering guidance

This notice is intended to preserve attribution. It does not assert ownership over the original upstream work.
